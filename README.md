Sistema Integral de Atención al Usuario (SIAU) - Gestión de PQRS

➦ PQRS-backend

Backend del Sistema Integral de Atención al Usuario orientado a la gestión de Gestión de
quejas, reclamos y sugerencias, permitiendo el registro, seguimiento y administración de solicitudes de manera estructurada.


➦ Objetivo General

Desarrollar un sistema backend para la gestión de Peticiones, Quejas, Reclamos y Sugerencias (PQRS), que permita registrar, administrar y dar seguimiento a las solicitudes de los usuarios de manera organizada y eficiente.

➦ Objetivos Específicos
Diseñar e implementar una base de datos estructurada que garantice la correcta gestión y trazabilidad de las solicitudes.
Desarrollar funcionalidades para el registro, consulta y actualización del estado de las PQRS.
Permitir la gestión de usuarios y roles dentro del sistema.
Implementar el manejo de archivos adjuntos asociados a las solicitudes.
Optimizar los procesos de atención mediante una estructura clara y centralizada de la información.


➦  Funcionalidades principales

* Registro de solicitudes PQRS
* Gestión de estados (pendiente, en proceso, resuelto)
* Respuestas a solicitudes
* Trazabilidad de cada caso
* Gestión de usuarios
* Manejo de archivos adjuntos

➦   Tecnologías

* PostgreSQL / Supabase
* HTML
* CSS
* JS
* reCATPCHA
* Docker
* Express + Node

➦   Estado del proyecto

En desarrollo – Fase de estructuración y mejora del modelo de base de datos.
